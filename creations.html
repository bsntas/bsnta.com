UNO

Browser 2–6 Players P2P

A real-time multiplayer UNO card game that runs entirely in the browser — no server, no accounts, no install. Share a room code and play from any device. Includes UNO Flip mode with dual-sided dark-side rules.

Tambola

Browser Host + Player Offline

A Tambola / Housie number draw simulator covering 1–90. The host draws numbers and verifies winning claims; players generate a ticket and mark off numbers as they’re called. Works fully offline once loaded.

Bingo

Browser 2–5 Players P2P

A real-time multiplayer Bingo game that runs in the browser with no server needed. Host a room with a 4-character code, pick a grid from 5×5 up to 15×15, and take turns calling numbers until someone completes 5 lines.
